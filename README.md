# Anti-Frontrunner

Privacy-first domain search. All lookups run in your browser via DNS-over-HTTPS—no backend, no search logging.

**Production URL:** [privatedomainsearch.com](https://privatedomainseach.com)

## Local preview

```bash
npm install
npm run build
npx --yes serve . -p 3000
```

Open [http://localhost:3000](http://localhost:3000).


## Affiliate links

Registrar affiliate IDs live in `index.html` (`AFFILIATE_IDS`). Partners typically require a live site before approval—deploy first, then add IDs.
